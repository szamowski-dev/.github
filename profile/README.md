## Principles

| We prefer | Because |
| --- | --- |
| Native apps over wrappers | Platform details matter. |
| Fast software over bloated software | Responsiveness compounds. |
| Focused products over feature overload | Useful tools should stay sharp. |
| Design systems over random UI | Consistency creates trust. |
| Shipping over endless planning | Real feedback beats internal certainty. |
